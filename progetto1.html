Armadio Smart - Sistema di gestione degli outfit

Questo è un progetto basato su Python e Flask per la gestione degli outfit in un armadio smart. Il progetto consente agli utenti di creare outfit utilizzando capi di abbigliamento già salvati, come pantaloni, maglie e scarpe.

Come iniziare

Per iniziare, clonare il progetto sulla propria macchina locale usando il seguente comando:

git clone https://github.com/mrfavios/smart-wardrobe.git

Una volta clonato il progetto, spostatevi nella cartella del progetto e installate i pacchetti necessari usando i seguenti comandi:

  • pip install Flask
  • pip install json
  • pip install socket

Successivamente, è necessario salvare le immagini degli articoli di abbigliamento nelle cartelle appropriate della directory static/img. Le immagini devono essere salvate nelle seguenti cartelle:

  • static/img/pantaloni
  • static/img/camicie
  • static/img/scarpe

Il progetto viene fornito con outfit di esempio, che si trovano nel file outfit.json.

Per eseguire il progetto, aprire una finestra di terminale e navigare nella directory del progetto. Quindi, eseguire il seguente comando:

python app.py

In questo modo si avvierà l'applicazione Flask e si potrà accedere al sistema di guardaroba smart aprendo il browser web e navigando verso http://(il vostro ip locale):5000.

Come funziona

Il sistema di guardaroba intelligente consente agli utenti di visualizzare i propri capi di abbigliamento e di creare outfit selezionando gli articoli da ogni categoria. Gli utenti possono anche salvare gli outfit per riferimenti futuri.

Il progetto utilizza Python e Flask per il backend e HTML, CSS e JavaScript per il frontend.
